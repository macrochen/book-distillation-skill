# book-distillation-skill

把一本非虚构书、同一作者的一系列非虚构书，或同一人的同一思想在不同时期/不同版本中的书籍，蒸馏成可复用方法而非普通摘要的 Skill。

## 适用场景
- 把一本非虚构书提炼成专题 Skill
- 把同一作者多本非虚构书提炼成作者操作系统
- 把同一人的同一思想在不同年份、不同版本、不同整理本中的表达蒸成稳定内核 + 演化轨迹
- 从书摘、章节摘要、读书笔记中恢复出判断框架、证据偏好、表达骨架和边界条件

## 这个 Skill 关注什么
它关注的不是“这本书讲了什么”，而是：
- 它在解决什么问题
- 它如何观察问题
- 它如何下判断
- 它如何组织表达
- 它在哪些情况下失效

## 目录
- `SKILL.md`：主流程与判断规则
- `templates/single-book-distillation-template.md`：单本非虚构书蒸馏模板
- `templates/author-series-distillation-template.md`：同一作者多本非虚构书蒸馏模板
- `templates/thought-version-distillation-template.md`：同一思想多版本蒸馏模板

## 推荐产物
- 一本非虚构书 → 专题 Skill
- 同一作者多本非虚构书 → Author Core + Work Skill + Persona + Domain Modules + Corrections
- 同一思想多版本材料 → Stable Core + Version Map + Drift Map + Usage Rules

## 使用建议
1. 先确认是一本文献还是同一作者多本文献。
2. 优先使用一手材料，二手材料只做补充。
3. 先做局部卡片，再统计重复项，再处理冲突。
4. 最终产物必须包含边界条件；没有边界就不算完成蒸馏。

## 注意
- 本 Skill 只处理非虚构类书籍与相关整理材料。
- 不要把全文堆进 SKILL.md。
- 不要把语气模仿当成作者能力。
- 不要把版本整理差异直接当成思想演化。
- 材料不全时，必须标注覆盖率和置信度。
